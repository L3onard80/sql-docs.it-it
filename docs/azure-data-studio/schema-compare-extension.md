# <a name="schema-compare-extension-preview"></a>Estensione Confronto schema (anteprima)
L'estensione Confronto schema consente di confrontare facilmente file e database con estensione DACPAC e applicare le modifiche dall'origine alla destinazione.

Questa funzionalità è attualmente disponibile in versione di anteprima iniziale. Segnalare eventuali problemi e richieste di funzionalità [qui](https://github.com/microsoft/azuredatastudio/issues).

## <a name="install-the-schema-compare-extension"></a>Installare l'estensione Confronto schema

1. Per aprire Gestione estensioni e accedere alle estensioni disponibili, selezionare l'icona delle estensioni oppure l'opzione **Estensioni** dal menu **Visualizza**.
2. Selezionare un'estensione disponibile per visualizzarne i dettagli.
1. Selezionare l'estensione desiderata (Confronto schema) e **installarla**.

## <a name="how-do-i-start-a-schema-comparison"></a>Come è possibile avviare l'estensione Confronto schema?
* Il punto di ingresso principale di Confronto schema è fare clic con il pulsante destro del mouse su un database in Esplora oggetti e quindi scegliere **Confronto schema**.
* È anche possibile avviare la finestra di dialogo Confronto schema dal riquadro comandi (CTRL+MAIUSC+P) cercando **Confronto schema**

## <a name="why-would-i-use-the-schema-compare"></a>Perché usare Confronto schema?
L'estensione Confronto schema è stata creata per aggiungere la possibilità di confrontare gli schemi di file e database DACPAC e applicare le modifiche necessarie.
