# <a name="schema-compare-extension-preview"></a>Estensione Confronto schema (anteprima)
L'estensione del confronto Schema offre un'esperienza facile da usare per confrontare i database e i file con estensione dacpac e applicare le modifiche dall'origine alla destinazione.

Questa esperienza è attualmente la versione di anteprima iniziale. Segnalare problemi e richieste di funzionalità [qui.](https://github.com/microsoft/azuredatastudio/issues)

## <a name="install-the-schema-compare-extension"></a>Installare l'estensione del confronto Schema

1. Per aprire la gestione delle estensioni e accedere alle estensioni disponibili, selezionare l'icona delle estensioni o scegliere **Estensioni** dal menu **Visualizza**.
2. Selezionare un'estensione disponibile per visualizzarne i dettagli.
1. Selezionare l'estensione desiderata (confronto Schema) e **installare** è.

## <a name="how-do-i-start-a-schema-comparison"></a>Come si inizia a un confronto Schema?
* Il punto di ingresso principale per confronto schema consiste nel fare doppio clic su un database in Esplora oggetti e fare clic su **confronto Schema**.
* L'utente può anche avviare la finestra di dialogo Confronto schema dal riquadro comandi (Ctrl + MAIUSC + P) eseguendo una ricerca per **confronto Schema**

## <a name="why-would-i-use-the-schema-compare"></a>Il motivo per cui è necessario utilizzare confronto Schema?
Confronto schema è stato creato per aggiungere la possibilità di confrontare gli schemi dal file con estensione dacpac e database e applicare le modifiche.
