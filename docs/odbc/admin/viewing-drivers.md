# <a name="viewing-drivers"></a>Visualizzazione dei driver
Per configurare le origini dati, è necessario avere installato almeno un driver nel sistema. Il **driver** scheda le **Amministrazione origine dati ODBC** nella finestra di dialogo sono elencati tutti i driver installati nel computer, inclusi il nome, versione, azienda, nome file e data di creazione di file di ogni driver. Per aggiungere o eliminare un driver dal sistema, è necessario utilizzare il programma di installazione del driver. Per altre informazioni su come modificare i driver, vedere [gestione di origini dati](../../odbc/admin/managing-data-sources.md).
