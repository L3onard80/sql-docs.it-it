# <a name="machine-data-sources"></a>Origini dati del computer
*Origini dati del computer* vengono archiviate nel sistema con un nome definito dall'utente. Tutte le informazioni associate nome dell'origine dati è la necessità di gestione Driver e il driver per la connessione all'origine dati. Per un'origine dati Xbase, potrebbe trattarsi del nome del driver Xbase, il percorso completo della directory contenente i file Xbase e alcune opzioni che indicano i driver come usare tali file, ad esempio la modalità utente singolo o sola lettura. Per un'origine dati Oracle, ciò potrebbe essere il nome del driver di Oracle, il server in cui si trova il sistema DBMS di Oracle, SQL * Net stringa di connessione che identifica il codice SQL\*driver .NET da usare e l'ID del sistema del database nel server.
