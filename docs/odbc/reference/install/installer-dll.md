# <a name="installer-dll"></a>DLL di installazione
> [!NOTE]  
>  A partire da Windows XP e Windows Server 2003, ODBC è incluso nel sistema operativo Windows. È consigliabile installare solo in modo esplicito ODBC nelle versioni precedenti di Windows.  
  
 La DLL del programma di installazione contiene funzioni per l'installazione e la rimozione di componenti ODBC, per la gestione delle informazioni sul Registro di sistema e per la gestione delle informazioni sulle origini dati. Viene scritto da Microsoft e può essere ridistribuito. Per una descrizione completa delle funzioni nella DLL del programma di installazione, vedere informazioni di [riferimento sull'API DLL del programma di installazione](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
