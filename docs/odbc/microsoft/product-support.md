# <a name="product-support"></a>Supporto tecnico clienti
Il supporto tecnico per ODBC è fornito dal servizio supporto tecnico Microsoft. Poiché molti prodotti Microsoft utilizzano ODBC come componente principale e ridistribuiscono inoltre i driver ODBC, il servizio supporto tecnico Microsoft considera ODBC come parte di ogni prodotto che incorpora ODBC come componente. Il supporto per ODBC viene fornito in conformità con il contratto di supporto del prodotto. Per ulteriori informazioni, vedere la documentazione del singolo prodotto.
