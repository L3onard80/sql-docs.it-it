# <a name="odbc-errors"></a>Errori ODBC
Quando si verifica un errore, il driver di Database Desktop ODBC Microsoft restituiscono il numero di errori nativi, il valore SQLSTATE (un codice di errore ODBC) e un messaggio di errore.  
  
 In questa sezione vengono trattati gli argomenti seguenti.  
  
-   [Errore nativo](../../odbc/microsoft/native-error.md)  
  
-   [SQLSTATE](../../odbc/microsoft/sqlstate.md)  
  
-   [Messaggi di errore](../../odbc/microsoft/odbc-jet-error-messages.md)  
  
-   [Errori aritmetici](../../odbc/microsoft/arithmetic-errors.md)
