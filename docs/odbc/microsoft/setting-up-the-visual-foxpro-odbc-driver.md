# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>Configurazione del driver ODBC Visual FoxPro
Per informazioni su come scaricare e installare il driver ODBC Visual FoxPro, vedere [Driver ODBC Visual FoxPro](https://go.microsoft.com/fwlink/?LinkId=121318) sul download di MSDN.  
  
 Utilizzare il programma di installazione di Microsoft Visual FoxPro ODBC Driver per eseguire le operazioni seguenti:  
  
-   Aggiungere nuovi componenti.  
  
-   Rimuovere i componenti installati.  
  
-   Ripetere l'installazione per ripristinare le impostazioni e i file mancanti.  
  
-   Rimuovere tutti i componenti installati in precedenza.  
  
 Dopo aver installato il driver nel sistema, il programma di installazione riconosce i componenti installati driver e vengono visualizzate le finestre di dialogo aggiuntive che consentono di modificare la configurazione del driver.
