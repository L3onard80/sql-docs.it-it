# <a name="convert-function-limitations"></a>Limitazioni della funzione CONVERT
Tipo di risultato di errori di conversione nella colonna interessata viene impostata su NULL.  
  
 Tipo di dati data né il TIMESTAMP può essere convertito a un altro data tipo (o se stessa) per la funzione CONVERT.
