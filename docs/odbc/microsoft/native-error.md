# <a name="native-error"></a>Errore nativo
Codici di errore nativo positivi vengono generati dai driver di Database Desktop. Codici di errore negativi vengono generati da Microsoft Jet. Per altre informazioni, vedere la *Guida per programmatori di Microsoft Jet Database Engine*.
