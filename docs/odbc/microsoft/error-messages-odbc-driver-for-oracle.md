# <a name="error-messages-odbc-driver-for-oracle"></a>Messaggi di errore (driver ODBC per Oracle)
> [!IMPORTANT]  
>  Questa funzionalità verrà rimossa in una versione futura di Windows. Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata. In alternativa, usare il driver ODBC fornito da Oracle.  
  
 Quando si verifica un errore, il Driver ODBC di Microsoft per Oracle restituisce il valore SQLSTATE (un codice di errore ODBC) e un messaggio di errore. Il driver deriva queste informazioni dagli errori rilevati dal driver e gli errori restituiti dal Server Oracle.
