# <a name="odbc-driver-for-oracle-conformance-levels"></a>Livelli di conformità del driver ODBC per Oracle
> [!IMPORTANT]  
>  Questa funzionalità verrà rimossa in una versione futura di Windows. Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata. In alternativa, usare il driver ODBC fornito da Oracle.  
  
 ODBC definisce due tipi di standard di conformità per i driver: la conformità di API standard e la conformità di grammatica SQL standard. Conformità API fa riferimento alle funzioni che supporta un driver. Conformità SQL si intende la grammatica SQL supportate dal driver. Ogni standard di conformità è costituita da livelli.  
  
 Questa sezione vengono fornite informazioni di conformità riguardanti il Driver ODBC per Oracle.  
  
 In questa sezione vengono trattati gli argomenti seguenti.  
  
-   [Livello di conformità API (Driver ODCB per Oracle)](../../odbc/microsoft/api-conformance-level-odbc-driver-for-oracle.md)  
  
-   [Opzioni supportate (Driver ODCB per Oracle)](../../odbc/microsoft/supported-options-odbc-driver-for-oracle.md)  
  
-   [Livelli di conformità SQL (Driver ODCB per Oracle)](../../odbc/microsoft/sql-conformance-levels-odbc-driver-for-oracle.md)
