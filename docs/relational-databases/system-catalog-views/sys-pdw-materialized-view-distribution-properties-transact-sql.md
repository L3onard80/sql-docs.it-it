# <a name="syspdwmaterializedviewdistributionproperties-transact-sql-preview"></a>Sys.pdw_materialized_view_distribution_properties (Transact-SQL) (anteprima)

[!INCLUDE[tsql-appliesto-xxxxxx-xxxx-asdw-xxx-md](../../includes/tsql-appliesto-xxxxxx-xxxx-asdw-xxx-md.md)]

Visualizza le informazioni di distribuzione viste materializzate.  
  
|Nome colonna|Tipo di dati|Descrizione|  
|-----------------|---------------|-----------------| 
|object_id|**int**|ID della vista materializzata per le tre proprietà sono state specificate.| 
|distribution_policy |**tinyint**|2 = HASH</br>4 = ROUND_ROBIN|  
|distribution_policy_desc |**nvarchar(60)**|HASH, ROUND_ROBIN|  
 
## <a name="permissions"></a>Permissions

È richiesta l'autorizzazione VIEW DATABASE STATE.
