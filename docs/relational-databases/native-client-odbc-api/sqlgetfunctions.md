# <a name="sqlgetfunctions"></a>SQLGetFunctions
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  L'interfaccia del driver ODBC di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client è conforme ai tre livelli di conformità definiti per ODBC (ISO, Open Group e ODBC Level 2). **SQLGetFunctions** segnala che tutte le funzioni ODBC sono supportate per il driver odbc di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client.
