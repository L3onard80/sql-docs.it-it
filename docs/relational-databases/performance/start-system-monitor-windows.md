# <a name="start-system-monitor-windows"></a>Avvio di Monitoraggio di sistema (Windows)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  Monitoraggio di sistema consente di monitorare l'utilizzo delle risorse di sistema Raccogliere e visualizzare dati sulle prestazioni in tempo reale sotto forma di contatori. Ãˆ possibile usare contatori per risorse server quali l'uso del processore e della memoria e per molte risorse di Microsoft [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], come blocchi e transazioni.  
  
## <a name="start-system-monitor-in-windows"></a>Avviare Monitoraggio di sistema in Windows  
  
Scegliere **Esegui** dal menu **Start**, digitare **perfmon** nella finestra di dialogo **Esegui** e quindi fare clic su **OK**.
