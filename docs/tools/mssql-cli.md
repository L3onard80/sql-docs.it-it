# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>strumento di query della riga di comando MSSQL-CLI per SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

MSSQL-CLI è uno strumento da riga di comando interattivo per eseguire query SQL Server, installarlo in Windows, macOS o Linux.

## <a name="install-mssql-cli"></a>Installare MSSQL-CLI

Per istruzioni dettagliate sull'installazione, vedere la [Guida all'installazione](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md)o, se si conosce PIP, installare eseguendo il comando seguente:

```$ pip install mssql-cli```
