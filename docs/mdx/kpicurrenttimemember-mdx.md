# <a name="kpicurrenttimemember-mdx"></a>KPICurrentTimeMember (MDX)


  Restituisce il membro temporale corrente dell'indicatore di prestazioni chiave (KPI) specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
KPICurrentTimeMember(KPI_Name)  
```  
  
## <a name="arguments"></a>Argomenti  
 *KPI_Nam*e  
 Espressione stringa valida che specifica il nome dell'indicatore di prestazioni chiave (KPI).  
  
## <a name="remarks"></a>Note  
 Il membro temporale di un indicatore KPI pu√≤ essere diverso dal membro predefinito della dimensione temporale.
