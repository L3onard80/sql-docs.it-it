La classe PDO contiene metodi che consentono all'applicazione PHP di connettersi a un'istanza di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
PDO {}  
```
