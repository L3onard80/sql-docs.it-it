# <a name="xml-data-in-sql-server"></a>Dati XML in SQL Server

![Download-DownArrow-Circled](../../../ssdt/media/download.png)[Scaricare ADO.NET](../../sql-connection-libraries.md#anchor-20-drivers-relational-access)

SQL Server espone la funzionalit√† di SQLXML all'interno di .NET. Gli sviluppatori possono scrivere applicazioni che accedono ai dati XML da un'istanza di SQL Server, trasmettono i dati nell'ambiente .NET, elaborano i dati e inviano gli aggiornamenti al SQL Server. I dati XML possono essere usati in vari modi in SQL Server, ad esempio nell'archiviazione dei dati e come valori di parametro per il recupero dei dati. La classe **SqlXml** in .NET fornisce il supporto sul lato client per l'uso dei dati archiviati in una colonna XML in SQL Server. Per ulteriori informazioni, vedere "SQLXML Managed Classes" in documentazione online di SQL Server.
