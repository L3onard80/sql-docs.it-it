# <a name="getbinarystream-method-sqlservercallablestatement"></a>Metodo getBinaryStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come flusso binario di byte senza interruzioni.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[getBinaryStream &#40;int&#41;](../../../connect/jdbc/reference/getbinarystream-int.md)|Recupera il valore del parametro designato come flusso binario di byte senza interruzioni in base all'indice del parametro.|  
|[getBinaryStream &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getbinarystream-java-lang-string.md)|Recupera il valore del parametro designato come flusso binario di byte senza interruzioni in base al nome del parametro.|
