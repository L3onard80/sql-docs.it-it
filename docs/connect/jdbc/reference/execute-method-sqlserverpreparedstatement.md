# <a name="execute-method-sqlserverpreparedstatement"></a>Metodo execute (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Esegue l'istruzione SQL in questo oggetto [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md), che può essere qualsiasi tipo di istruzione SQL.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|nome|Descrizione|  
|----------|-----------------|  
|[execute ()](../../../connect/jdbc/reference/execute-method.md)|Esegue l'istruzione SQL in questo oggetto [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md), che può essere qualsiasi tipo di istruzione SQL.|  
|[execute (java.lang.String)](../../../connect/jdbc/reference/execute-method-java-lang-string.md)|Esegue l'istruzione SQL specificata, che può restituire più risultati.|
