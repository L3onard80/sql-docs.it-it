# <a name="position-method-sqlservernclob"></a>Metodo position (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera la posizione dei caratteri dell'oggetto **NClob** specificato o della sottostringa in **NClob** in base alla posizione di inizio specificata.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[Metodo position &#40;java.sql.NClob, long&#41;](../../../connect/jdbc/reference/position-method-java-sql-nclob-long.md)|Recupera la posizione del carattere in corrispondenza della quale viene visualizzato l'oggetto **NClob** specificato *searchstr* nell'oggetto **NClob**.|  
|[Metodo position &#40;java.lang.String, long&#41; &#40;SQLServerNClob&#41;](../../../connect/jdbc/reference/position-method-java-lang-string-long-sqlservernclob.md)|Recupera la posizione del carattere in cui viene visualizzata la sottostringa specificata *searchstr* nel valore **NCLOB** rappresentato da questo oggetto **NClob**.|
