# <a name="position-method-sqlservernclob"></a>Metodo position (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera la posizione dei caratteri dell'oggetto **NClob** specificato o della sottostringa in **NClob** in base alla posizione di inizio specificata.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|nome|Descrizione|  
|----------|-----------------|  
|[Metodo position &#40;java.sql.NClob, long&#41;](../../../connect/jdbc/reference/position-method-java-sql-nclob-long.md)|Recupera la posizione del carattere in corrispondenza del quale l'oggetto specificato **NClob** oggetto *searchstr* viene visualizzato in questo **NClob** oggetto.|  
|[Metodo Position &#40;lang, lungo&#41; &#40;SQLServerNClob&#41;](../../../connect/jdbc/reference/position-method-java-lang-string-long-sqlservernclob.md)|Recupera la posizione del carattere in corrispondenza del quale la sottostringa specificata *searchstr* viene visualizzato nei **NCLOB** valore rappresentato da questo **NClob** oggetto.|
