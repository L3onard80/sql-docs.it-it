# <a name="getapplicationintent-method-sqlserverdatasource"></a>Metodo getApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce il valore della **applicationIntent** proprietà di connessione.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce il valore della **applicationIntent** proprietà di connessione.  
  
## <a name="remarks"></a>Remarks  
 Per altre informazioni sul **applicationIntent** proprietà di connessione, vedere [impostando le proprietà di connessione](../../../connect/jdbc/setting-the-connection-properties.md).
