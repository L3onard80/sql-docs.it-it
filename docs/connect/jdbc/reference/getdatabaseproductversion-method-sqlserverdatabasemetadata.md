# <a name="getdatabaseproductversion-method-sqlserverdatabasemetadata"></a>Metodo getDatabaseProductVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il numero di versione di questo prodotto di database.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.lang.String getDatabaseProductVersion()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Valore **String** contenente il numero di versione del prodotto di database.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Questo metodo getDatabaseProductVersion viene specificato dal metodo getDatabaseProductVersion nell'interfaccia DatabaseMetaData.
