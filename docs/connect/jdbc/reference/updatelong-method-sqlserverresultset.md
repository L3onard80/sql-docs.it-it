# <a name="updatelong-method-sqlserverresultset"></a>Metodo updateLong (SQLServerResultSet)

|Nome|Descrizione|  
|----------|-----------------|  
|[updateLong (int, long)](../../../connect/jdbc/reference/updatelong-method-int-long.md)|Aggiorna la colonna designata con un valore **long** in base all'indice della colonna.|  
|[updateLong (java.lang.String, long)](../../../connect/jdbc/reference/updatelong-method-java-lang-string-long.md)|Aggiorna la colonna designata con un valore **long** in base al nome della colonna.|
