# <a name="getnstring-method-sqlservercallablestatement"></a>Metodo getNString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro **nchar**, **nvarchar**o **LONGNVARCHAR** designato come stringa nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|nome|Descrizione|  
|----------|-----------------|  
|[Metodo &#40;getNString int&#41;](../../../connect/jdbc/reference/getnstring-method-int.md)|Recupera il valore del parametro **nchar**, **nvarchar**o **LONGNVARCHAR** designato come stringa nel linguaggio di programmazione Java.|  
|[Metodo getNString &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getnstring-method-java-lang-string.md)|Recupera il valore del parametro **nchar**, **nvarchar**o **LONGNVARCHAR** designato come stringa nel linguaggio di programmazione Java.|
