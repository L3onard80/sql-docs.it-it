# <a name="setapplicationintent-method-sqlserverdatasource"></a>Metodo setApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il valore della proprietà di connessione **ApplicationIntent** .  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>Parametri  
 *applicationIntent*  
  
 Nuovo valore della proprietà di connessione **ApplicationIntent** .  
  
## <a name="remarks"></a>Remarks  
 Per ulteriori informazioni sulla proprietà di connessione **ApplicationIntent** , vedere [impostazione delle proprietà di connessione](../../../connect/jdbc/setting-the-connection-properties.md).
