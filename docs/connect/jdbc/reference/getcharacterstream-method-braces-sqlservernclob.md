# <a name="getcharacterstream-method--sqlservernclob"></a>Metodo getCharacterStream () (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera i dati **NCLOB** come oggetto **Reader** o come flusso di caratteri.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.io.Reader getCharacterStream()  
```  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="return-value"></a>Valore restituito  
 Oggetto Reader contenente i dati **NCLOB**.  
  
## <a name="remarks"></a>Remarks  
 Questo metodo getCharacterStream viene specificato dal metodo getCharacterStream nell'interfaccia java. SQL. NClob.
