# <a name="getstring-method-sqlservercallablestatement"></a>Metodo getString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come oggetto **String** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[getString (int)](../../../connect/jdbc/reference/getstring-method-int.md)|Recupera il valore del parametro designato come oggetto **String** nel linguaggio di programmazione Java in base all'indice del parametro.|  
|[getString (java.lang.String)](../../../connect/jdbc/reference/getstring-method-java-lang-string.md)|Recupera il valore del parametro designato come oggetto **String** nel linguaggio di programmazione Java in base al nome del parametro.|
