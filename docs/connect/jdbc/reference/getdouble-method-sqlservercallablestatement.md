# <a name="getdouble-method-sqlservercallablestatement"></a>Metodo getDouble (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come oggetto **double** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|nome|Descrizione|  
|----------|-----------------|  
|[getDouble (int)](../../../connect/jdbc/reference/getdouble-method-int.md)|Recupera il valore del parametro designato come oggetto **double** nel linguaggio di programmazione Java in base all'indice del parametro.|  
|[getDouble (java.lang.String)](../../../connect/jdbc/reference/getdouble-method-java-lang-string.md)|Recupera il valore del parametro designato come oggetto **double** nel linguaggio di programmazione Java in base al nome del parametro.|
