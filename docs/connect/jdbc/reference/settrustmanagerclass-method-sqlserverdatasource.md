# <a name="settrustmanagerclass-method-sqlserverdatasource"></a>Metodo setTrustManagerClass (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il valore di stringa della proprietà di connessione le proprietà di connessione.
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public void setTrustManagerClass(java.lang.String trustManagerClass)  
```  
  
#### <a name="parameters"></a>Parametri  
 *trustManagerClass*  
  
 Oggetto **stringa** che contiene il nome completo della classe di un javax.net.ssl.TrustManager personalizzato.
