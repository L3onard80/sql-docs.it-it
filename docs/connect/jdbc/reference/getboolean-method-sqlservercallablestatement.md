# <a name="getboolean-method-sqlservercallablestatement"></a>Metodo getBoolean (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come valore **booleano**.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|nome|Descrizione|  
|----------|-----------------|  
|[getBoolean (int)](../../../connect/jdbc/reference/getboolean-method-int.md)|Recupera il valore del parametro designato come valore **booleano** in base all'indice del parametro.|  
|[getBoolean (java.lang.String)](../../../connect/jdbc/reference/getboolean-method-java-lang-string.md)|Recupera il valore del parametro designato come valore **booleano** in base al nome del parametro.|
