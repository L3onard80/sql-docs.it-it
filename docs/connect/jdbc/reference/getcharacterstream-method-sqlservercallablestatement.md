# <a name="getcharacterstream-method-sqlservercallablestatement"></a>Metodo getCharacterStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come oggetto java.io.Reader.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|nome|Descrizione|  
|----------|-----------------|  
|[getCharacterStream &#40;int&#41;](../../../connect/jdbc/reference/getcharacterstream-int.md)|Recupera il valore del parametro designato come oggetto java.io.Reader in base all'indice del parametro.|  
|[getCharacterStream &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getcharacterstream-java-lang-string.md)|Recupera il valore del parametro designato come oggetto java.io.Reader in base al nome del parametro.|
