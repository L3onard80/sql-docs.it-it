# <a name="deploying-sql-server-2016-powerpivot-and-power-view-in-sharepoint-2016"></a>Distribuzione di SQL Server 2016 Power Pivot e Power View in SharePoint 2016
[!INCLUDE[ssas-appliesto-sqlas](../../../includes/ssas-appliesto-sqlas.md)]
  **Riepilogo:** Questo white paper offre agli amministratori di SharePoint e architetti istruzioni dettagliate per la distribuzione e configurazione di un ambiente demo di Microsoft BI basato su versioni di anteprima di SharePoint Server 2016, Office Online Server e SQL Stack di Business Intelligence Server 2016 per SharePoint 2016. Dopo una breve introduzione che illustra le modifiche importanti apportate all'architettura e alle dipendenze di sistema corrispondenti, descrive i requisiti software e di configurazione e fornisce un percorso di distribuzione consigliato per abilitare e verificare le funzionalità di Business Intelligence in tre fasi principali. Questo white paper presenta anche i problemi noti esistenti nelle versioni SharePoint Server 2016 Beta 2, Office Online Server Preview e SQL Server 2016 CTP 3.1 e suggerisce le soluzioni alternative appropriate. Queste soluzioni alternative non saranno necessarie nelle versioni finali dei prodotti. In fase di distribuzione delle versioni RTM verrà pubblicata una versione aggiornata di questo white paper.
