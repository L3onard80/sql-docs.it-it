# <a name="changelog-for-the-report-viewer-controls"></a>Registro modifiche per i controlli Visualizzatore report

Questo registro modifiche registra il rilascio dei controlli Visualizzatore report Windows Form e Web Form.

## <a name="15900148"></a>15.900.148
Con questo aggiornamento
 - Correzione di un bug che impedisce il caricamento dei report senza parametri mediante Server.LoadReportDefinition

Controllo Visualizzatore report [Web Form](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Webforms/150.900.148)
 - Supporta l'incorporamento nelle pagine da destra a sinistra (pagine che cambiano il flusso del testo usando la proprietà CSS *direction: rtl;*).
 - Supporta la personalizzazione del testo della finestra di dialogo Stampa tramite l'interfaccia di localizzazione *IReportViewerMessages5*.
 - Migliorato il supporto dell'accessibilità.

Controllo Visualizzatore report [Windows Form](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Winforms/150.900.148)
 - Correzione della stampa quando un'app è in esecuzione in una modalità con valori DPI alti.
