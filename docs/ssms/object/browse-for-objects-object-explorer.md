## <a name="options"></a>Opzioni  
**Casella di controllo**  
Selezionare la casella di controllo per includere l'oggetto.  
  
**Icona**  
Icona che rappresenta il tipo di oggetto.  
  
**Nome**  
Nome dell'oggetto .  
  
**Tipo**  
Tipo di oggetto di database.
