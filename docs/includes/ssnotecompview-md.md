  Questa tabella di sistema di SQL Server 2000 è disponibile come vista per compatibilità con le versioni precedenti. È consigliabile usare invece le viste di sistema correnti di SQL Server. Per trovare la vista o le viste di sistema equivalenti, vedere [Mapping di tabelle di sistema a viste di sistema &#40;Transact-SQL&#41;](../relational-databases/system-tables/mapping-system-tables-to-system-views-transact-sql.md). Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata.
