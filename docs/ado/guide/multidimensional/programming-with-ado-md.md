# <a name="programming-with-ado-md"></a>Programmazione con ADO MD
Per utilizzare ADO MD con uno strumento di sviluppo, è necessario stabilire un riferimento alla libreria dei tipi ADO MD. La descrizione della libreria ADO MD è Microsoft ActiveX Data Objects (multidimensionale) Library. Il nome file della libreria ADO MD è msadomd e l'ID di programma (ProgID) è "ADOMD". Per altre informazioni su come stabilire i riferimenti alle librerie, vedere la documentazione dello strumento di sviluppo.
