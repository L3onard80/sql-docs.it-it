# <a name="source-property-ado-md"></a>Proprietà Source (ADO MD)
Indica l'origine per i dati presenti il [cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Imposta o restituisce un **Variant**, ed è lettura/scrittura per chiuso [Cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) oggetti e read-only per aprire **Cellset** oggetti. Il **Variant** deve contenere un valore valido **stringa**, ad esempio, una query MDX.
