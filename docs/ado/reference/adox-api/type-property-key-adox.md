# <a name="type-property-key-adox"></a>Proprietà Type (Key) (ADOX)
Indica il tipo della chiave.  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Imposta o restituisce un **lungo** valore che può essere uno delle [KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md) costanti. Il valore predefinito è **adKeyPrimary**.  
  
## <a name="remarks"></a>Note  
 Questa proprietà è di sola lettura sul [chiave](../../../ado/reference/adox-api/key-object-adox.md) già aggiunti a una raccolta di oggetti.
