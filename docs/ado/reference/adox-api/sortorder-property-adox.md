# <a name="sortorder-property-adox"></a>Proprietà SortOrder (ADOX)
Indica la sequenza di ordinamento per la colonna (solo colonne di indice).  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Imposta e restituisce un **lungo** valore che può essere uno delle [SortOrderEnum](../../../ado/reference/adox-api/sortorderenum.md) costanti. Il valore predefinito è **adSortAscending**.  
  
## <a name="remarks"></a>Note  
 Questa proprietà si applica solo a [colonna](../../../ado/reference/adox-api/column-object-adox.md) gli oggetti nel [colonne](../../../ado/reference/adox-api/columns-collection-adox.md) raccolta di un [indice](../../../ado/reference/adox-api/index-object-adox.md).
