# <a name="adox-code-examples-in-microsoft-visual-basic"></a>Esempi di codice ADOX in Microsoft Visual Basic

> [!NOTE]
>  Incollare l'intero esempio di codice, da sub a End Sub, nell'editor di codice. L'esempio potrebbe non essere eseguito correttamente se si utilizzano esempi parziali o se la formattazione del paragrafo va persa.
