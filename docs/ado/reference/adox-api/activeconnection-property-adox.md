# <a name="activeconnection-property-adox"></a>Proprietà ActiveConnection (ADOX)
Indica l'oggetto ADO [Connection](../../../ado/reference/ado-api/connection-object-ado.md) oggetto a cui il [catalogo](../../../ado/reference/adox-api/catalog-object-adox.md) appartiene.  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Set di una **Connection** oggetto o una **stringa** contenente la definizione per una connessione. Restituisce attivo **connessione** oggetto.  
  
## <a name="remarks"></a>Note  
 Il valore predefinito è un riferimento a un oggetto null.
