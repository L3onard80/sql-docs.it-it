# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
Specifica il numero di record da recuperare da un [Recordset](../../../ado/reference/ado-api/recordset-object-ado.md).  
  
|Costante|Value|Descrizione|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|Recupera il resto dei record del **Recordset**, dalla posizione corrente o un segnalibro specificato dalle *avviare* parametro del [GetRows](../../../ado/reference/ado-api/getrows-method-ado.md) (metodo).|

|--------------|
