# <a name="ado-syntax-indexes"></a>Indici sintassi ADO
La sintassi per chiamare proprietà e metodi ADO varia a seconda dell'ambiente di sviluppo. Il resto dei riferimenti al linguaggio ADO utilizza il linguaggio di programmazione Microsoft Visual Basic per illustrare la sintassi di metodi e le proprietà di ADO. Tuttavia, vedere le sezioni seguenti per esempi di sintassi più specifici in base al linguaggio di programmazione e metodologia:  
  
-   Il [ADO per indice sintassi di Visual C++ per COM](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-for-com.md) descrive metodi e proprietà ADO senza utilizzare il **#import** direttiva del compilatore Microsoft Visual c++.  
  
-   Il [ADO per indice sintassi di Visual C++ con #import](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-with-sharpimport.md) tratta ADO proprietà e metodi quando si usa la **#import** direttiva del compilatore Microsoft Visual c++.
