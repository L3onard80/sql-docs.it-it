# <a name="ado-collections"></a>Raccolte di ADO
Le relazioni tra queste raccolte e gli oggetti ADO sono rappresentate nel [modello a oggetti ADO](../../../ado/reference/ado-api/ado-object-model.md).  
  
 Ogni raccolta può contenere l'oggetto corrispondente. Ad esempio, un' [errore](../../../ado/reference/ado-api/error-object.md) oggetto può essere contenuto in un [errori](../../../ado/reference/ado-api/errors-collection-ado.md) raccolta. Per altre informazioni sugli oggetti, vedere [gli oggetti ADO](../../../ado/reference/ado-api/ado-objects-and-interfaces.md) o un argomento di oggetto specifico.  
  
|||  
|-|-|  
|[errori](../../../ado/reference/ado-api/errors-collection-ado.md)|Contiene tutti i **errore** gli oggetti creati in risposta a un singolo errore relativi al provider.|  
|[Fields](../../../ado/reference/ado-api/fields-collection-ado.md)|Contiene tutti i **campo** gli oggetti di un **Recordset** oggetto.|  
|[Parametri](../../../ado/reference/ado-api/parameters-collection-ado.md)|Contiene tutti i **parametri** gli oggetti di un **comando** oggetto.|  
|[Proprietà](../../../ado/reference/ado-api/properties-collection-ado.md)|Contiene tutti i **proprietà** gli oggetti per un'istanza specifica di un oggetto.|
