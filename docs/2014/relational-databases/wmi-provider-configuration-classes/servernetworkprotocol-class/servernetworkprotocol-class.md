# <a name="servernetworkprotocol-class"></a>Classe ServerNetworkProtocol
  Il servernetworkprotocol [classe ServerNetworkProtocol]-class.md) rappresenta i protocolli di rete installati in un'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] che consentono la comunicazione con i client in una rete.  
  
 Con la servernetworkprotocol delle [classe ServerNetworkProtocol]-class.md), è possibile:  
  
-   Controllare nome, versione, file con estensione dll, numero di proprietà e nome visualizzato del protocollo di rete del server a cui si fa riferimento.  
  
-   Fare riferimento a proprietà e indirizzi IP del protocollo di rete.  
  
-   Abilitare e disabilitare il protocollo di rete del server.
