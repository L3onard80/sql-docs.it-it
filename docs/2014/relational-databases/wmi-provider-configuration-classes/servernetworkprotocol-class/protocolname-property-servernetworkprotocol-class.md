# <a name="protocolname-property-servernetworkprotocol-class"></a>Proprietà ProtocolName (classe ServerNetworkProtocol)
  Ottiene il nome del protocollo di rete del server.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 A [classe ServerNetworkProtocol](servernetworkprotocol-class.md) che rappresenta il protocollo di rete utilizzato dall'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Stringa che specifica il nome del protocollo di rete del server.
