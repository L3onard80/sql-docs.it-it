# <a name="localdb_error_unknown_version"></a>LOCALDB_ERROR_UNKNOWN_VERSION
    
## <a name="details"></a>Dettagli  
  
|||  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|269|  
|Origine evento|Runtime database locale di SQL Server 12.0|  
|Componente|API di Runtime database locale|  
|Testo del messaggio|Versione del database locale specificata non disponibile nel computer.|  
  
## <a name="explanation"></a>Spiegazione  
 Versione specificata non disponibile.  
  
## <a name="user-action"></a>Azione dell'utente  
 Installare la versione di Runtime database locale richiesta.
