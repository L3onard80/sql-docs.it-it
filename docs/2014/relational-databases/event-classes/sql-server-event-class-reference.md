# <a name="sql-server-event-class-reference"></a>Guida di riferimento alla classe di evento SQL Server

Gli eventi registrati corrispondono a istanze delle classi di evento nella definizione della traccia. In [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]le classi di evento e le rispettive categorie di eventi sono disponibili nella scheda **Selezione eventi** della finestra di dialogo **Proprietà file di traccia** .
