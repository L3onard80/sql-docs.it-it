# <a name="select-missing-dimension-keys-cube-wizard"></a>Seleziona chiavi di dimensione mancanti (Creazione guidata cubo)
  Utilizzare la pagina **Seleziona chiavi di dimensione mancanti** per selezionare le colonne chiave da utilizzare quando una nuova dimensione non ha chiavi definite per la tabella primaria.  
  
## <a name="options"></a>Opzioni  
 **Chiavi dimensione**  
 Selezionare una o più colonne chiave dalla tabella primaria della dimensione che saranno utilizzate come chiave della dimensione.  
  
 Se vengono selezionate più colonne chiave, sarà creata una chiave composta.
