# <a name="how-to-export-reports"></a>Procedura: Export Reports
  Il Visualizzatore report di Preparazione aggiornamento consente di esportare una versione in formato file flat del report. In questo modo è possibile utilizzare le informazioni in un foglio di calcolo o in un documento di elaborazione testi oppure inviare le informazioni ad altri utenti tramite posta elettronica.  
  
### <a name="to-export-a-report"></a>Per esportare un report  
  
1.  Nel visualizzatore di report aprire il report da esportare. Per istruzioni, vedere [come: Visualizzare un Report di preparazione aggiornamento](../../../2014/sql-server/install/how-to-view-an-upgrade-advisor-report.md).  
  
2.  Per filtrare il report, selezionare un valore compreso il **Filtra per** casella.  
  
3.  Fare clic su **esportare Report**.  
  
4.  Nel **file di rapporto di esportazione** della finestra di dialogo digitare un nome file, scegliere una destinazione e quindi fare clic su **OK**. Il file viene salvato in formato CSV (valori delimitati da virgole). È possibile utilizzare Microsoft Excel o applicazioni simili per aprire il file. Se si desidera un altro formato per i dati, è possibile copiare le informazioni da Excel in Microsoft Word o in altre applicazioni.
