# <a name="how-to-validate-successful-installation-of-a-sql-server-update"></a>Procedura: Verificare la corretta installazione di un aggiornamento di SQL Server
  Per convalidare la corretta installazione di un aggiornamento di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]:  
  
-   Verificare che i servizi [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] siano online.  
  
-   Verificare le informazioni di versione restituite dal @@version query.
