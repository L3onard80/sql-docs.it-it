# <a name="nsltservice-namegt-properties-advanced-tab"></a>Proprietà NS$&lt;nome servizio&gt; (scheda Avanzate)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] viene implementato come un servizio denominato nel formato **NS$**_< nome_configurato_da_utente >_. Se sono state definite proprietà personalizzate, verranno visualizzate in questa scheda con i rispettivi valori. È possibile che questa scheda non contenga proprietà.
