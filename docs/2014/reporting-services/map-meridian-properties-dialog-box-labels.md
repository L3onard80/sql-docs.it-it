# <a name="map-meridian-properties-dialog-box-labels"></a>Finestra di dialogo Proprietà meridiani mappa, Etichette
  Utilizzare la finestra di dialogo **Proprietà MapMeridian** per modificare le opzioni delle etichette per la griglia verticale nel viewport mappa. Un meridiano rappresenta il valore seguente a seconda del sistema di coordinate specificato per il viewport:  
  
-   **Planar**. La coordinata Y.  
  
-   Area **geografica**. La longitudine della proiezione corrente.  
  
 Fare clic sul pulsante **Espressione** (*fx*) per modificare un'espressione che imposta il valore per l'opzione.  
  
## <a name="options"></a>Opzioni  
 **Intervallo**  
 Digitare un valore intero in gradi che specifica l'intervallo tra meridiani. Per impostazione predefinita, è selezionato **Automatico** . **Auto** indica che il valore viene determinato automaticamente in base ai dati spaziali.  
  
 **Mostra etichette**  
 Selezionare questa opzione per visualizzare le etichette per i meridiani.  
  
 **Posizione**  
 Selezionare una posizione per visualizzare le etichette relativamente alla posizione superiore, centrale e inferiore del viewport. La posizione predefinita è **Vicino**.  
  
-   **Vicino a** Visualizzare le etichette sul bordo sinistro.  
  
-   **Un trimestre** Visualizzare le etichette a metà strada tra il bordo sinistro e il centro.  
  
-   Al **centro** Visualizzare le etichette al centro.  
  
-   **Tre trimestri** Visualizza le etichette a metà tra il centro e il bordo destro.  
  
-   A **lungo** Visualizzare le etichette sul bordo destro.
