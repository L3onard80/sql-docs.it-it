# <a name="deploying-an-exception-message-box-application"></a>Distribuzione di un'applicazione finestra di messaggio eccezione
  Finestra di messaggio eccezione viene installata da [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] ed è supportato per l'uso nelle applicazioni Windows personalizzate per migliorare la gestione delle eccezioni. Poiché la finestra di messaggio eccezione viene installata in tutte le edizioni di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] eccetto [!INCLUDE[ssEW](../../includes/ssew-md.md)], è possibile usarlo senza alcuna configurazione aggiuntiva in qualsiasi computer in cui [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] componenti client, tra cui il SDK, sono stati installati.
